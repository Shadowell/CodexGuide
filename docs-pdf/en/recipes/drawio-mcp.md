# Codex × Draw.io MCP: AI automatically draws architecture diagrams

As the saying goes, a picture is worth a thousand words. When we want to introduce a project architecture, business process, or system module relationship, or the core logic of daily reading, if we only use words to explain it, readers will easily get tired of reading. If we can organize it into an information graph or flow chart, the cost of understanding will be reduced a lot.

**Draw.io** is a very good tool. It can be used to draw:

1. flow chart
2. Architecture diagram
3. mind map
4. System design drawing
5. business process diagram
6. technology roadmap

Many programmers, product managers, and architects use it to express the relationships between complex systems. Some time ago, Draw.io officially released its own MCP.

GitHub address: https://github.com/jgraph/drawio-mcp

![image-20260512192851857](../images/image-20260512192851857.png)

This chapter introduces how to use Codex to install and use Draw.io's MCP tool to draw information diagrams or architecture diagrams.

## 2. Get started

I choose the content of a book and let Draw.io draw a Chinese infographic based on the core ideas of the book.

![image-20260512193130155](../images/image-20260512193130155.png)

You can request it at will and customize the content, theme, style and other related requirements to be drawn.

> **Tip:** In the prompt word, be sure to emphasize using **Draw.io's MCP** to complete the task, so that it will call this tool.

![image-20260512193232787](../images/image-20260512193232787.png)

After the drawing is completed, it will summarize the task and **automatically open the browser to display the drawing results**.
