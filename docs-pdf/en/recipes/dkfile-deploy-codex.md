# Codex × DKFile: Publish AI web pages to the public network with one click

Many people are stuck with the same problem with HTML web pages made with AI: **The web page is only local and cannot be opened when sent to friends**.

There are many solutions for people with programming experience - GitHub Pages, Vercel, Cloudflare Pages - but these tools are not friendly to the average user: What is a branch? What is deployment? What is domain name resolution?

This article introduces a tool that is extremely friendly to novices: **[DKFile](https://dkfile.net)**.

---

## Advantages of DKFile

- **Minimalist operation:** Upload the HTML generated by AI, and DKFile will automatically complete all hosting work
- **One-click generation:** Click to upload and automatically obtain the public network access link
- **No configuration required:** No need to push code, fill in configuration, or know any deployment knowledge

Operation process:

1. Click "Upload File" and select the local HTML file
2. Click "Upload" and the system will automatically complete the hosting
3. After the upload is completed **directly obtain the public network access link** and you can share it with anyone

## 2. Codex + DKFile API fully automated

Manual uploading still requires opening the website, finding the file, and clicking a button. In fact, this process can be completely automated by Codex.

DKFile provides API documentation. The overall process is:

1. Let Codex read DKFile's API documentation
2. After Codex completes the web page production, it automatically calls the API to upload the file.
3. Return directly to the public network access link

**You don’t need to open a browser or upload manually during the entire process, it’s all automated. **

### specific steps

First let Codex read the API documentation, it will tell you what you need to prepare (mainly your **API Key**):

![image-20260513120426087](../images/image-20260513120426087.png)

Click "API docking document" in the DKFile backend and follow the instructions to create your own API Key (if you don't understand, just ask Codex and it will guide you).

Then send your API Key and design requirements to Codex, for example:

```
Help me design an HTML webpage for Obsidian novice tutorials,
After the design is completed, use DKFile API (Key: xxx) to automatically upload and return the public network link.
```

Codex will automatically complete the design, call the API to upload, and finally tell you the file location and **directly accessible public network link**:

![image-20260513120544640](../images/image-20260513120544640.png)
