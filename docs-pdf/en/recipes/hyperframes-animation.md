# Codex × HyperFrames: Generate animated videos with code

**HyperFrames** and **Remotion** which were very popular some time ago are both coded video generation:

1. Remotion: Write videos using React
2. HyperFrames: Make videos using HTML, CSS or JS

If you want to make some lightweight videos, such as converting web pages to videos, converting documents to videos, etc., then using **HyperFrames** is a very good choice.

---

## 1. Install the plugin

Open the left sidebar of the Codex Desktop App and find "**Plugins and Skills**". After entering, search for **HyperFrames** to find the plugin and install it directly.

![image-20260512215014826](../images/image-20260512215014826.png)

> HyperFrames also have corresponding **Skill** (skills) that can be used by both. You can choose the appropriate or favorite way to install it.

---

## 2. How to use

I directly gave it the content of a book, and then asked it to make a video based on the content of the book, and it would call this skill to complete the corresponding task.

![image-20260512215253160](../images/image-20260512215253160.png)

After the task is completed, it will tell us **Return the address of the local file**, click to jump to view the specific content.

![image-20260512215544769](../images/image-20260512215544769.png)

If you look closely, you'll find that it actually generates several images in JPEG format and then assembles the images into a video. It's probably such a process.

![image-20260512215633692](../images/image-20260512215633692.png)

The video generated for the first time may not be as perfect as imagined, and you need to continue to increase the demand.

For example, if you conduct multiple iterations on the following aspects, the effect may be better:

1. Screen pertableance
2. The specific effects generated
3. Specific details such as copywriting, color matching, and background for each page

This requires our patience and step-by-step attempts.

![image-20260512215719224](../images/image-20260512215719224.png)

> **Tips:** If you have a target video, you can directly send it the finished video effect and let it imitate it for production. The effect may be more in line with the requirements.

If you are interested in how Remotion makes videos, then I recommend the skill of Remotion, an open source project shared by Teacher Wang on GitHub. You can learn it as a reference.
