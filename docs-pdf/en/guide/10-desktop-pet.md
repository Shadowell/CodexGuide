# How to set up your own Codex desktop pet

Some time ago, Codex added a **desktop pet** that will accompany you in your work. It's not just a decoration, it actually shows what Codex is currently working on.

The greatest value of this pet is the visualization of status:

1. **Task progress is clear at a glance:** You can see the progress of the current task without having to switch back to the Codex interface all the time.
2. **Real-time status feedback:**
   - It displays a busy screen when the Codex is busy.
   - A reminder will be sent when your confirmation is required.
   - After the task is completed, it will also let you know that you can check the results.

---

## 1. Installation

Open "**Settings**" → "**Appearance**" and scroll down to the "**Pet**" column. Here you can select your favorite pet type and click "**Wake**", and the selected pet will appear on the desktop.

![image-20260512224500231](../images/image-20260512224500231.png)

**Shortcut command method:** If you don't want to enter the settings, you can also directly enter `/` in the chat box and select the "**Pet**" option to wake up or put away the desktop pet.

![image-20260512225501451](../images/image-20260512225501451.png)

---

## 2. Use

After waking up, give the command as usual and then do other things. The little pet will accompany you on the screen and display the progress of task completion in real time.

For example, the following situation: it is searching and pertableing our tasks.

![image-20260512225715200](../images/image-20260512225715200.png)

After the task is completed, a **green check** will be displayed to indicate that the task has been completed. In other words, we can see its execution process and completion results in real time, without staring at the Codex window and waiting.

![image-20260512230004092](../images/image-20260512230004092.png)
