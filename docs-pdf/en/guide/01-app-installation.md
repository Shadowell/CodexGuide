# Codex Desktop App Download and Installation

::: tip final check
The last date of verification of official information: 2026-05-27. The download address and installation method are subject to [OpenAI Codex product page](https://openai.com/codex/) and [chatgpt.com/codex/cloud](https://chatgpt.com/codex/cloud). The available functions may differ in different regions and account packages.
:::

The **Codex Desktop App** in this tutorial refers to the computer client, not the mobile App. It is officially released by OpenAI and supports macOS and Windows. After installation, you can manage projects locally, initiate tasks, and use Skills and Automations.

## download

Open [chatgpt.com/codex/cloud](https://chatgpt.com/codex/cloud), and the download button for the corresponding system will be displayed in the center of the page.

**macOS：**

![Codex Desktop App Download Page macOS](../images/image-20260511172444574.png)

Click "Download for macOS" to download the `.dmg` installation package.

**Windows：**

![Codex Desktop App Download Page Windows](../images/image-20260511172634781.png)

Windows users click the corresponding download button, run the installation program after the download is complete, and follow the prompts to complete the installation.

## macOS installation steps

1. Open the downloaded `.dmg` file
2. Drag the Codex icon into the Applications folder
3. When starting up for the first time, macOS may prompt "Unable to verify developer." Go to "System Settings → Privacy and Security" and click "Open Anyway"

## Log in

After the installation is complete, open the App and log in with your ChatGPT / OpenAI account.

::: warning Pay attention to package restrictions
The full functionality of the Codex Desktop App (multi-agent parallelism, Skills, Automations, etc.) requires the **Plus and above package**. After logging in with a free account, some functions will be limited or unavailable. If you need to upgrade, refer to the next chapter: [Subscribe to ChatGPT Plus](./02-subscribe-plus.md).
:::

::: tip
If the login fails, you can consider switching agent nodes or switching agents. You can choose a non-US node.
:::

## Post-install verification

After successful login, the left column should display the two entrances "Projects" and "Chats", and the current account information should be displayed at the top. If the interface loads normally, the installation is successful.
